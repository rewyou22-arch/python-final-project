## 📌 Описание проекта

Консольное приложение на Python, которое симулирует работу **склада товаров**:

- ведение учета товаров и их количества
- возможность автоматического пополнения существующих позиций
- контроль остатков при списании
- удаление товаров из базы при нулевом остатке

---

## 🚀 Функциональность

- **Добавить товар**: 
  - если товар с таким названием уже есть, его количество увеличится
  - если товара нет, он будет создан как новая позиция
- **Списать товар**: 
  - проверка наличия товара на складе
  - контроль того, чтобы нельзя было списать больше имеющегося количества
  - при полном списании (остаток 0) товар удаляется из списка
- **Показать склад**: вывод полного списка товаров с указанием их названия и текущего количества
- **Поиск**: регистронезависимый поиск товаров по названию

## 🧭 Как пользоваться

После запуска программы доступно текстовое меню:

- **1** — добавить товар (нужно ввести название и количество)
- **2** — списать товар (указать название и сколько штук забрать)
- **3** — показать склад (выведет все текущие товары)
- **4** — выход

При добавлении товара, если вы введете название, которое уже есть (например, "Яблоки"), программа не создаст новый пункт, а просто прибавит количество к уже существующим "Яблокам".

## 📁 Структура проекта

- `main.py` — основной файл с бесконечным циклом меню и логикой взаимодействия с пользователем
- `warehouse.py` — класс `Warehouse`, отвечающий за управление списком товаров (добавление, списание, поиск)
- `product.py` — класс `Product`, описывающий отдельную единицу товара (имя, количество, вывод инфо)
